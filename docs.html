Events

These vaules should be set where neccesary

Click events can be triggered with HTML by using: data-event="{{Tealium UDO}}" data-value="{{Button Value}}" OR in JavaScript
utag_data['event_name:{{Tealium UDO}}'] = {{ Value }};

Page View events can be triggerd by using: utag_data['event_name:{{Tealium UDO}}'] = true;

Checkout Products

Otherwise known as s.products, these vaules should be set where neccesary in a checkout process.

These are arrays. When pushing an item to an array, the values must line up across each of these items.

Product values & events can be populated / triggerd by using: utag_data.{{Tealium UDO}}.push(value);
