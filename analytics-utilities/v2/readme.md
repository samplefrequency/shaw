# Analytics Utilities
> Utilities for collecting, enriching, and distributing data to analytics & marketing cloud platfomrs.

## Installation
```sh
npm install shaw-analytics --save
```

## Development setup

In the package directory, use the console commands:

```sh
npm install
gulp
```
